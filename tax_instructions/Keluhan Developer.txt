Keluhan Developer
1. Untuk data CSV sangat sulit dikembangkan di class Group. Ketika di import data CSV sebagai parameter treatment, data CSV hanya bisa membaca 1 data. Dalam arti untuk CSV dengan contoh data CSV 98,91,100, maka output datanya yang keluar hanya 98 dan itu semua Player juga mendapatkan output yang sama seperti yang saya contohkan tadi. Jadi sangat sulit dikembangkan untuk di distribusikan secara acak dan tidak bisa di kalkulasi. Jika data CSV tersebut di import dengan menggunakan class Player, outputnya bisa sesuai dengan baris data yang tersedia. Akan tetapi, di class Player ini juga memiliki logika yang cukup signifikan berbeda dengan class Group. Di class Player, data CSV bisa di distribusikan secara acak dan dapat digunakan untuk kalkulasi hanya saja di class Player dia akan mengambil data di player terakhir di masing-masing Group untuk menjadi penentu nilai kalkulasi Group. Dari semua proses pengembangan apps nya, INI YANG PALING CUKUP BIKIN FRUSTASI.
